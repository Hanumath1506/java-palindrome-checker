# Java Palindrome Checker

A simple Java program that checks whether a given string is a palindrome.

## Description
This program takes a string input and determines whether it reads the same forwards and backwards. It demonstrates basic string manipulation and control flow in Java.

## Technologies
- Java
- Object-Oriented Programming

## File Structure
src/
└── PalindromeCheck.java


## How to Run
Compile:
```bash
javac src/PalindromeCheck.java

```Run:
java src.PalindromeCheck
